Finance Enterprise System
A Python-based Finance Enterprise System that simulates basic banking operations such as account management, deposits, withdrawals, money transfer, fixed deposits, transaction history, and financial reports.
Features
	•	User Signup and Login
	•	Password Protection using getpass
	•	OTP Verification
	•	Deposit Money
	•	Withdraw Money
	•	Transfer Money with Limits
	•	Fixed Deposit (FD) Creation
	•	FD Withdrawal
	•	Transaction History
	•	Save Transactions to File
	•	Read Saved Transactions
	•	Financial Reports

Technologies Used
	•	Python 3
	•	Object-Oriented Programming (OOP)
	•	File Handling
	•	Datetime Module
	•	Random Module

How to Run the Project
1. Clone the Repository
git clone https://github.com/yourusername/finance-enterprise-system.git
2. Open Project Folder
cd finance-enterprise-system
3. Run the Python Program
python3 finance_system.py
